# 腾讯客户端开发公开课作业二   《UMG界面》作业
要求：
- 为目前的Demo添加游戏界面：包含角色个人信息，基础操作按钮，计分显示。
- 可选：添加一个得分排行榜，排行数据可使用SaveGame保存于本地，得分进入榜单时可编辑自己在榜单上显示的昵称。
